## 6. Punjabi-Specific Learning Modules

### 6.1 Gurmukhi Script Mastery

#### Phase 1: Letter Recognition (Weeks 1-2)
- **Uraa group:** ੳ (oora), ਅ (aira), ੲ (eeri) — the three base forms
- **Consonants:** 35 letters organized by mouth position
- **Vowels:** 10 independent vowels + matra signs
- **Activities:**
  - Letter flashcards with pronunciation
  - "Find the letter" games
  - Letter-sound matching
  - Animated stroke order videos

#### Phase 2: Writing Practice (Weeks 3-4)
- **Dotted-line tracing:** Guided stroke practice
- **Freehand writing:** AI recognition checks correctness
- **Letter formation rules:**
  - Top line (header) importance
  - Proper proportions
  - Connection rules for conjuncts
- **Gamified:** "Perfect stroke" bonuses, handwriting leaderboard

#### Phase 3: Reading Fluency (Weeks 5-8)
- **Syllable blending:** Combine consonants + vowels
- **Word decoding:** Simple 2-3 letter words
- **Sight words:** Common words for instant recognition
- **Graded readers:** Short Punjabi stories with audio

#### Phase 4: Advanced Orthography (Weeks 9-12)
- **Conjunct consonants:** ੍ (halant) usage
- **Nasalization:** ਂ (bindi) vs ਁ (tippi)
- **Gemination:** Double consonants
- **Punctuation:** । (danda) usage

### 6.2 Tone Recognition & Production

**The Punjabi Tone System**
Punjabi is a tonal language with three tones:
- **Low tone (grave):** `ਕੋੜਾ` kora = leper
- **Mid tone (level):** `ਕੋੜਾ` kora = horse (in some dialects)
- **High tone (acute):** `ਕੋੜਾ` kora = whip

#### Teaching Methodology

1. **Perception Training:**
   - Minimal pair exercises (same word, different tones)
   - Visual tone curves overlaid on audio waveforms
   - "Tone matching" game: listen and select correct tone curve

2. **Production Training:**
   - Record user pronunciation
   - AI analysis: pitch contour extraction
   - Visual feedback: user's pitch curve vs native speaker
   - "Tone accuracy score" (0-100)

3. **Contextual Practice:**
   - Sentences where tone changes meaning
   - "Save the sentence" game: wrong tone = funny/wrong meaning

### 6.3 Shahmukhi Script Module

**For Pakistani Punjabi & Urdu-script learners:**
- Parallel alphabet: ا, ب, پ, ت, ٹ, ث...
- Right-to-left writing practice
- Gurmukhi to Shahmukhi transliteration tool
- "Script switch" toggle in all lessons
- Content differences: Pakistani vocabulary, Islamic cultural context

### 6.4 Cultural Immersion Modules

#### Module: "A Day in Punjab"
- Morning: Wake up, farm routine, breakfast vocabulary
- Noon: Market visit, bargaining, food ordering
- Evening: Family time, TV, dinner
- Night: Gossip, dreams, bedtime phrases

#### Module: "The Punjabi Wedding"
- Pre-wedding: Roka, sagai, mehndi vocabulary
- Wedding day: Anand Karaj, baraat, jago
- Post-wedding: Doli, reception, blessings
- Interactive: Plan a virtual wedding, learn phrases for each event

#### Module: "Bhangra & Music"
- Learn lyrics to famous songs (licensed or public domain)
- Dance vocabulary: thumka, jhumar, gidha
- Instrument names: dhol, tumbi, algoza
- "Karaoke mode": Sing along with lyrics in Gurmukhi

#### Module: "Gurbani Basics" (Optional, Respectful)
- Mool Mantar learning
- Common Gurbani phrases
- Pronunciation for prayer
- Historical context (not religious instruction)
- Toggle: Available for interested learners, never forced

#### Module: "Diaspora Life"
- "Talking to grandparents": Respectful speech, family terms
- "At the Gurdwara": Etiquette, common phrases
- "Punjabi in Canada/UK": Diaspora slang, code-switching
- "Heritage identity": Discussing being Punjabi abroad

### 6.5 Dialect Awareness

**Major Punjabi Dialects Covered:**
| Dialect | Region | Key Features |
|---------|--------|--------------|
| **Majhi** | Amritsar, Lahore | Standard, taught in app |
| **Malwai** | Ludhiana, Ferozepur | Distinct vowel sounds |
| **Doabi** | Jalandhar, Hoshiarpur | Unique vocabulary |
| **Pothwari** | Rawalpindi area | Heavy Urdu influence |
| **Dogri** | Jammu area | Related, some overlap |

**Implementation:**
- All core content in Majhi (standard)
- "Regional flavor" badges: Learn 10 Malwai phrases
- Audio samples from different regions
- "Dialect detective" mini-game: Identify the dialect

## 7. Gamification System

**XP Sources:**
| Action | XP Reward |
|--------|-----------|
| Complete lesson | 10-20 XP |
| Perfect lesson (no mistakes) | +5 XP bonus |
| First attempt correct | +2 XP |
| Streak bonus (7+ days) | 2x XP |

| Review session | 5 XP |
| Speaking exercise | 10 XP |
| Writing exercise | 10 XP |
| Story completion | 25 XP |
| Challenge win | 20-50 XP |
| Community help | 10 XP |

**Levels:**
- 1-10: Seedling
- 11-25: Sprout
- 26-50: Sapling
- 51-100: Tree
- 101-200: Orchard
- 201-500: Garden
- 500+: Punjab

### 7.2 Streak System

- **Daily goal:** User sets 5, 10, 15, or 20 minutes
- **Streak freeze:** 1 free, then purchase with gems or watch ad
- **Streak repair:** Restore broken streak within 48 hours
- **Streak society:** Join streak groups for mutual accountability
- **Streak milestones:**
  - 7 days: "One Week Strong" badge
  - 30 days: "Month Master" badge + profile frame
  - 100 days: "Century Club" badge + exclusive avatar
  - 365 days: "Punjabi Legend" badge + real-world certificate

### 7.3 Currency: Gems & Coins

**Gems (Premium Currency):**
- Earned: Hard achievements, challenges, referrals
- Used: Streak freezes, bonus lessons, exclusive avatars, hint reveals

**Coins (Regular Currency):**
- Earned: Daily lessons, reviews, perfect scores
- Used: Village builder items, avatar customization, power-ups
